# 📚 Lenguajes de Programación - [Romar Palencia/307C1]

Este repositorio contiene los proyectos, prácticas y ejercicios realizados durante la asignatura de **Lenguajes de Programación**. El objetivo es explorar diferentes paradigmas y entender la sintaxis, semántica y ejecución de diversos lenguajes.

## 🛠️ Tecnologías Exploradas
* **Haskell / Lisp:** Paradigma Funcional.
* **Prolog:** Paradigma Lógico.
* **C++ / Java:** Paradigma Orientado a Objetos e Imperativo.
* **Python:** Scripting y multiparadigma.
